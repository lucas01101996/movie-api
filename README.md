# Movie API

RESTful API for managing a movie catalog, with pagination/search and basic authentication for admin operations.

## Tech Stack
- Java 21
- Spring Boot
- Spring Web (MVC)
- Spring Data JPA
- H2 (in-memory)
- Spring Security (Basic Auth)
- Bean Validation (jakarta.validation)

---

## How to Run

### Requirements
- Java 21
- Maven

### Run the application
```bash
mvn spring-boot:run
